# 理論

本章では、非線形自由表面波動問題に対するBEMソルバーの数学的定式化を解説します。

## アプローチ

本ソルバーは**ポテンシャル流れ理論**に基づいています。流体は非圧縮・非回転と仮定し、速度場はラプラス方程式を満たすスカラーポテンシャル $\phi$ の勾配で表されます。

$$\nabla^2 \phi = 0$$

流体領域全体を離散化する代わりに、**グリーンの定理**を用いて体積問題を領域境界 $\Gamma$ 上の境界積分方程式（BIE）に変換します。これにより空間次元が1つ減少し、境界面のみをメッシュ化すれば済みます。

境界面は**線形三角要素**で離散化されます。各時間ステップにおいて混合境界値問題を解きます。自由表面ではディリクレ条件（ $\phi$ 既知）、固体壁面および海底ではノイマン条件（ $\partial\phi/\partial n$ 既知）が課されます。

**高速多重極展開法（FMM）** により、BIE評価を $O(N^2)$ から $O(N)$ に高速化し、大規模3次元シミュレーションを実現します。線形方程式系はGMRESにより反復的に解かれます。

自由表面の位置とポテンシャルの時間積分には陽的ルンゲ・クッタ法を使用し、ALE（任意ラグランジュ・オイラー）メッシュ更新により要素品質を維持します。
