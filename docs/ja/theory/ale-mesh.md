# ALEメッシュ管理

本ページでは、ALE（Arbitrary Lagrangian--Eulerian、任意ラグランジュ・オイラー）法によるメッシュ管理手法について解説します。

## 動機

MEL法では自由表面をラグランジュ的に追跡しますが、浮体の運動や波の変形に伴いメッシュが歪み、計算精度の低下や破綻を引き起こします。特に以下の状況でメッシュ歪みが顕著になります。

- 浮体の大変位により浮体近傍のメッシュが引き伸ばされる
- 自由表面の変形によりメッシュが局所的に密集または疎になる
- 造波板の運動に追従する領域でメッシュが変形する

ALE法はラグランジュ法とオイラー法を組み合わせ、メッシュの運動を流体の運動から独立に制御することでこの問題を解決します。

## ALE定式化

ALE法では、メッシュ速度 $\mathbf{v}_m$ を流体速度 $\mathbf{u} = \nabla\phi$ とは独立に設定します。ラグランジュ法（ $\mathbf{v}_m = \mathbf{u}$ ）とオイラー法（ $\mathbf{v}_m = \mathbf{0}$ ）はそれぞれ特殊な場合です。

節点位置の更新は次式で行います。

$$\frac{d\mathbf{x}}{dt} = \mathbf{v}_m$$

## 修正物質微分

メッシュ速度と流体速度が異なるため、自由表面上の物理量の物質微分を修正する必要があります。

ラグランジュ微分（流体に追従した微分）$\frac{D}{Dt}$ とALE微分 $\frac{d}{dt}$ の関係は次式で与えられます。

$$\frac{D(\cdot)}{Dt} = \frac{d(\cdot)}{dt} + (\mathbf{u} - \mathbf{v}_m) \cdot \nabla(\cdot)$$

自由表面の力学的境界条件は次のように修正されます。

$$\frac{d\phi}{dt} = -gz + \frac{1}{2}|\nabla\phi|^2 + (\mathbf{v}_m - \nabla\phi) \cdot \nabla\phi$$

運動学的境界条件（鉛直方向）は次のようになります。

$$\frac{dz}{dt} = \frac{\partial\phi}{\partial z} + (\mathbf{v}_{m,x} - u)\frac{\partial z}{\partial x} + (\mathbf{v}_{m,y} - v)\frac{\partial z}{\partial y}$$

ここで $u, v$ はそれぞれ $x, y$ 方向の流速成分です。

## ディリクレ節点の再配置

自由表面上（ディリクレ条件）の節点は、メッシュ品質を維持するために再配置します。再配置にはラプラシアン平滑化を用います。

$$\mathbf{x}_i^{\text{new}} = \frac{1}{N_{\text{adj}}} \sum_{j \in \text{adj}(i)} \mathbf{x}_j$$

ここで $\text{adj}(i)$ は節点 $i$ に隣接する節点の集合、$N_{\text{adj}}$ はその数です。

実際には、完全にラプラシアン平滑化するのではなく、ラグランジュ的追跡とラプラシアン平滑化の重み付き平均を用います。再配置後は修正物質微分により物理量を更新します。

## ノイマン節点の補正

固体壁面上（ノイマン条件）の節点は、壁面上に拘束される必要があります。ALE法では以下の手順で処理します。

1. ラプラシアン平滑化により暫定的な位置を計算
2. 暫定位置を壁面上に射影（最近点探索）
3. 壁面形状に沿った方向のみメッシュ速度を許容

浮体表面上の節点は浮体の運動に追従させ、壁面の接線方向にのみ再配置を行います。

## メッシュ品質指標

メッシュ品質の評価には、外接円半径と内接円半径の比（CircumradiusToInradius ratio）を用います。

$$Q = \frac{R}{r}$$

ここで $R$ は三角要素の外接円半径、$r$ は内接円半径です。正三角形の場合 $Q = 2$ が最小値（最良品質）であり、$Q$ が大きいほど要素が歪んでいることを表します。

この指標をモニタリングすることで、メッシュ品質の劣化を検出し、必要に応じてリメッシュを実行します。
