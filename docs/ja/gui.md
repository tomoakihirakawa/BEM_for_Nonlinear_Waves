# GUI利用ガイド

GUIアプリケーションは、JSONを手動で記述せずにシミュレーション入力ファイルを作成・編集・エクスポートするためのビジュアルインターフェースです。

![GUIスクリーンショット](../images/gui_screenshot.png)

## 動作環境

| 要件 | バージョン | 備考 |
|------|-----------|------|
| Python | 3.11.x | 3.12以降は非対応（VTK互換性） |
| PySide6 | 6.9.3 | 6.10以降はmacOSでハング |
| macOS | 12以降 | Apple Silicon推奨 |

## インストールと起動

```bash
cd gui
sh run.sh
```

`run.sh`はPython仮想環境の作成、依存パッケージのインストール、アプリケーションの起動を自動で行います。

## ワークフロー：ケース選択からシミュレーション実行まで

基本的な流れは3ステップです：**ギャラリー** → **設定** → **出力**

### ステップ1 — ギャラリーからケースを選択

左パネルの**ギャラリー**タブに、利用可能なシミュレーションケースがカード形式で表示されます。

1. カードを見て、計算対象に最も近いケースを探す（例：水槽内の孤立波、波浪中の浮体など）
2. カードをクリックして読み込み

選択したケースの設定が、自動的に右隣の**設定**パネルに反映されます。

### ステップ2 — 設定でパラメータを編集

ケースを選択すると、ギャラリーの右側にある**設定**タブにシミュレーションパラメータが表示されます：

- **シミュレーション設定** — 時間刻み (`dt`)、終了時刻、ソルバーオプション
- **ALE設定** — メッシュ再配置手法、平滑化周期
- **要素タイプ** — 線形要素・二次要素の選択
- **オブジェクトツリー** — 全オブジェクト（水域、壁面、造波機、浮体など）を一覧表示し、表示/非表示を切り替え

パネル上部で**フォーム**表示（構造化エディタ）と**JSON**表示（生テキスト）を切り替えられます。

個々のオブジェクトのプロパティ（メッシュファイル、位置、質量、境界条件）を編集するには、オブジェクトツリーでオブジェクトを選択します。右側の**インスペクタ**パネルに詳細が表示され、フォーム/JSON両方の編集モードが使えます。

### ステップ3 — 入力ファイルを出力

設定が完了したら、**出力**セクションに移動します：

1. 出力先ディレクトリとファイル名を設定
2. **エクスポート**をクリックしてJSON入力ファイル（`settings.json`、`water.json`、`tank.json`など）を生成

### ステップ4 — ソルバーを実行

ターミナルから時間領域ソルバーを実行します。引数は出力ディレクトリです：

```bash
./main_time_domain /path/to/exported/output_directory/
```

ビルド方法については[はじめに](getting-started.html)を参照してください。

## ウィンドウ構成

### 中央 — 3Dビューポート

- メッシュ形状のインタラクティブ3D可視化
- 要素上にカーソルを合わせるとオブジェクト名とタイプを表示
- **フローティングツールバー**（下部中央） — ズーム、カメラリセット、軸表示
- **境界条件パネル**（右上） — 境界条件表示の切り替え

### 下部パネル

| タブ | 説明 |
|------|------|
| **コンソール** | コマンド実行ターミナル |
| **ログ** | タイムスタンプ付きログメッセージ（重要度レベル表示） |
| **出力ファイル** | 生成された出力ファイルの閲覧 |
| **履歴** | エクスポートジョブの履歴 |
| **チェックポイント** | シミュレーションスナップショットの管理 |
| **ブランチ** | 実験の分岐と比較 |

### ステータスバー

- 進捗インジケータ（エクスポート中に表示）
- 現在のケース名、オブジェクト数、ソルバータイプ
- **Toggle Theme** (Ctrl+T) — ダーク/ライトテーマの切り替え
- **EN/JP** (Ctrl+L) — インターフェース言語の切り替え
